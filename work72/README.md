# Mass Spring Cloth Simulation - work72

基于 Taichi 的质点弹簧布料模拟系统，支持多种弹簧类型和空间碰撞检测。

## 功能特性

### 三种弹簧类型

| 类型 | 连接方式 | 劲度系数 | 颜色 |
|------|----------|----------|------|
| **Structural（结构弹簧）** | 相邻质点（水平和垂直） | 8000 | 灰色 |
| **Shear（剪切弹簧）** | 对角线相邻质点 | 4000 | 橙色 |
| **Bending（弯曲弹簧）** | 间隔一个质点 | 2000 | 绿色 |

### 物理仿真

- **积分方法**: 显式欧拉、半隐式欧拉、隐式欧拉
- **物理参数**: 可调质量、阻尼系数、重力
- **稳定性**: 半隐式欧拉推荐用于日常使用

### 空间碰撞

- 球体碰撞检测
- 可调节球体半径
- 布料与球体的碰撞响应

## 运行要求

- Python 3.10 - 3.12
- Taichi >= 1.7.4
- 支持 GPU 加速（CUDA/Vulkan）或 CPU 回退

## 安装

```bash
pip install taichi
```

## 运行

```bash
python work72.py
```

## 操作说明

### 鼠标控制
- **右键拖拽**: 旋转视角
- **滚轮**: 缩放视角

### GUI 控制面板

**积分方法选择**
- Explicit Euler: 显式欧拉（易发散）
- Semi-Implicit Euler: 半隐式欧拉（稳定，推荐）
- Implicit Euler: 隐式欧拉（阻尼）

**弹簧类型开关**
- Structural Springs: 结构弹簧
- Shear Springs: 剪切弹簧
- Bending Springs: 弯曲弹簧

**碰撞设置**
- Sphere Collision: 球体碰撞开关
- Ball Radius: 调节球体半径

## 弹簧类型对比

### 仅结构弹簧
布料容易扭曲变形，缺乏剪切刚性。

### 结构 + 剪切弹簧
布料更稳定，保持矩形形状，抗剪切能力增强。

### 全部弹簧启用
布料具有更好的抗弯曲特性，形态更加自然。

## 效果展示

运行程序后可以看到布料从固定点垂落，与红色球体产生碰撞的效果。通过 GUI 可以实时调整各种参数观察变化。
