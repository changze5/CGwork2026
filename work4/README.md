# Work4 - 3D 光线投射与 Phong 光照模型

## 项目简介

本项目使用 Taichi 实现了一个基于光线投射（Ray Casting）技术的 3D 场景渲染系统，包含 Phong 光照模型和交互式参数调节功能。

## 功能特点

### 1. 几何物体
- **红色球体**：圆心坐标 (-1.2, -0.2, 0.0)，半径 1.2
- **紫色圆锥**：顶点 (1.2, 1.2, 0.0)，底面 y=-1.4，半径 1.2

### 2. 光照系统
- 摄像机位置：(0, 0, 5)
- 点光源位置：(2, 3, 4)，白色光源
- 实现完整的 Phong 光照模型：
  - 环境光（Ambient）
  - 漫反射（Diffuse）
  - 镜面高光（Specular）

### 3. 交互式控制面板
提供 4 个可调节参数的滑动条：
- Ka (环境光系数)：范围 0.0 ~ 1.0，默认值 0.2
- Kd (漫反射系数)：范围 0.0 ~ 1.0，默认值 0.7
- Ks (镜面高光系数)：范围 0.0 ~ 1.0，默认值 0.5
- N (Shininess 高光指数)：范围 1.0 ~ 128.0，默认值 32.0

## 技术实现

### 光线求交
- `intersect_sphere()`：光线与球体相交测试
- `intersect_cone()`：光线与圆锥相交测试
- 实现 Z-buffer 深度测试，确保正确的遮挡关系

### 光照计算
- `normalize()`：向量归一化
- `reflect()`：计算反射向量
- Phong 光照模型实现

## 运行环境

- Python 3.7 ~ 3.12
- Taichi 1.7.4 或更高版本

